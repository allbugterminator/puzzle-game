## 🎮 如何在Cocos Creator中运行

### 步骤1：打开项目
1. 启动 Cocos Creator 3.8.x
2. 选择 **文件** → **打开项目**
3. 选择 `puzzle-game` 文件夹
4. 等待项目加载完成

### 步骤2：打开游戏场景
1. 在 **资源管理器** 中找到 `assets/scenes/`
2. 双击 `Game.scene`
3. 场景将在编辑器中打开

### 步骤3：运行预览
1. 点击顶部工具栏的 **预览** 按钮（或按 Ctrl/Cmd + P）
2. 游戏将在浏览器中运行

## 🔧 常见问题排查

### 场景无法打开
1. 确认Cocos Creator版本为 3.8.x
2. 检查.scene文件JSON格式是否完整
3. 尝试重新导入项目

### 关卡数据加载失败
1. 确认JSON文件在 `assets/resources/levels/` 目录下
2. 检查文件名格式：`level_1_1.json`
3. 查看控制台错误信息

### 实体不显示
1. 检查EntityFactory是否正确挂载
2. 确认entityContainer节点已设置
3. 检查颜色值是否在有效范围(0-255)

## 📝 代码结构说明

### 核心系统关系
```
GameController (游戏控制器)
    ├── GridSystem (网格系统) - 坐标转换、渲染
    ├── PlayerController (玩家控制器) - 输入处理、移动
    ├── EntityFactory (实体工厂) - 创建实体
    └── RuleSystem (规则系统) - 规则解析、状态更新
```

### 数据流
```
关卡JSON → GameController → EntityFactory创建实体
                              ↓
                    PlayerController处理输入
                              ↓
                    RuleSystem更新规则状态
                              ↓
                    GameController检测胜利
```
