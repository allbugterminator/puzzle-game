# 微光迷途 - 2D微恐解谜游戏

一个受《Baba Is You》启发的2D微恐解谜游戏，采用Cocos Creator 3.8开发，支持微信小游戏平台。

## 项目概述

- **游戏类型**: 2D网格解谜 + 微恐氛围
- **核心机制**: Baba Is You风格的规则文字系统
- **技术栈**: Cocos Creator 3.8 + TypeScript
- **目标平台**: 微信小游戏

### 运行方式

#### 1. 开发环境要求

- Cocos Creator 3.8.x LTS
- Node.js 16+

#### 2. 打开项目

```bash
# 安装依赖
npm install

# 使用Cocos Creator打开项目
# 1. 打开Cocos Creator 3.8.x
# 2. 选择 "打开项目" -> 选择 puzzle-game 文件夹
# 3. 等待项目加载完成
```

#### 3. 运行游戏

1. 在Cocos Creator中打开 `Game.scene` 场景
2. 点击顶部工具栏的 **预览** 按钮
3. 游戏将在浏览器中运行

### 测试方法

#### 单元测试关卡

**1-1 欢迎来到迷雾**
- ✅ 测试四方向移动正常
- ✅ 测试墙壁阻挡碰撞
- ✅ 测试到达目标点触发胜利

**1-2 推推看**
- ✅ 测试推动单个方块
- ✅ 测试方块压上目标点过关
- ✅ 测试不能推动多个方块

**1-3 规则觉醒**
- ✅ 测试推动规则文字
- ✅ 测试"你+是+可推"规则生效
- ✅ 测试规则改变后玩家可被推动

**1-4 转换身份**
- ✅ 测试破坏"方块是停止"规则
- ✅ 测试组成"方块是你"规则
- ✅ 测试控制方块到达目标点

**1-5 胜利之路**
- ✅ 测试组成"你是胜利"规则
- ✅ 测试规则生效直接过关

### 测试方法

#### 单元测试关卡

每个关卡都有特定的测试目标：

**1-1 初醒**
- 测试移动：确保四个方向移动正常
- 测试碰撞：确保无法穿过墙体
- 测试胜利：到达出口触发胜利

**1-2 推动**
- 测试推动：可以推动可推块
- 测试连锁：推块到目标位置
- 测试阻挡：不能推动多个块

**1-3 规则初识**
- 测试规则文字：可以推动"你""是""可推"文字
- 测试规则生效：推动后规则改变游戏行为

**1-4 规则重构**
- 测试规则破坏：分离规则文字使规则失效
- 测试规则重组：组合新规则创造通路

**1-5 你就是胜利**
- 测试复杂规则：玩家接触"胜利"规则文字即胜利
- 测试多种解法：不同规则组合达成目标

## 视觉风格

### 色彩方案

- **背景色**: #1A1A2E (深蓝黑)
- **主角色**: #C8C8D0 (浅灰)
- **强调色**: #E94560 (暗红)
- **文字色**: #EAEAEA (浅白)

### 氛围效果

1. **暗角效果 (Vignette)**
   - 强度: 0.4
   - 柔和度: 1.2
   - 营造压抑氛围

2. **色调映射 (Color Grading)**
   - 低饱和度: 0.6
   - 冷色调偏移
   - 高对比度: 1.3

3. **光影效果**
   - 主角周围有微弱光晕
   - 出口有暗红色发光
   - 规则文字有边框高亮

## 性能优化

- **对象池**: 实体复用

## 微信小游戏发布

### 配置步骤

1. 在 `build-templates/wechatgame/project.config.json` 中填入AppID
2. 在Cocos Creator中选择 "项目" -> "构建发布"
3. 选择平台: 微信小游戏
4. 设置AppID
5. 点击构建

### 分包配置

```json
{
  "subpackages": [
    {"name": "level_pack_1", "root": "subpackages/level_pack_1/"},
    {"name": "level_pack_2", "root": "subpackages/level_pack_2/"},
    {"name": "level_pack_3", "root": "subpackages/level_pack_3/"},
    {"name": "level_pack_4", "root": "subpackages/level_pack_4/"}
```
